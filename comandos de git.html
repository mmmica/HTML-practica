Comandos: pwd (para saber donde estas parado) ls (para saber todo lo que tenemos) cd x/ (nombre de la carpeta), (para movernos entre carpetas) (con tab autocompleta la ruta que queremos escribir) clear (borrar consola) mkdir carpeta (nombre de la carpeta), (para crear carpetas) touch archivo.html (nombre del archivo), (para crear archivos) code . (para abrir en visual el archivo) LL (para listar mas informacion y muestra lo que contiene) rm archivo.html (nombre del archivo), (para borrar un archivo) rm -r x/ (nombre de la carpeta), (para borrar carpetas) cd .. (para volver para atras) cp archivo.html x (para copiar un archivo dentro de una carpeta) mv archivo.html x (para mover un archivo adentro de una carpeta) git status (muestra el status de la branch) git add . (para agregar todos los cambios)
