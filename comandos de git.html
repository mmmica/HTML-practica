Comandos: pwd (para saber donde estas parado) ls (para saber todo lo que tenemos) cd x/ (nombre de la carpeta), (para movernos entre carpetas) (con tab autocompleta la ruta que queremos escribir) clear (borrar consola) mkdir carpeta (nombre de la carpeta), (para crear carpetas) touch archivo.html (nombre del archivo), (para crear archivos) code . (para abrir en visual el archivo) LL (para listar mas informacion y muestra lo que contiene) rm archivo.html (nombre del archivo), (para borrar un archivo) rm -r x/ (nombre de la carpeta), (para borrar carpetas) cd .. (para volver para atras) cp archivo.html x (para copiar un archivo dentro de una carpeta) mv archivo.html x (para mover un archivo adentro de una carpeta) git status (muestra el status de la branch) git add . (para agregar todos los cambios) git commit -m "un mensaje" (para crear el paquete que despu√©s se va a subir), (cada vez que se hace un cambio, hay que ser lo mas claro posible en el mensaje del commit, para indicar con que se estuvo trabajando y que cambios se subieron) git restore . (elimina las lineas que no queremos mas del archivo) git branch -M main (para cambiar la rama a main si tenemos master) git remote add origin https://github.com/mmmica/HTML-practica.git (para conectar el git local con el repositorio) git push -u origin main (para pushear los archivos al repositorio de la nube)
